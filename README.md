# React Post Component 📝

תרגיל ב-React המציג רשימת פוסטים עם אפשרות לעריכת התוכן בזמן אמת.

## יכולות הפרויקט:
- **Props:** העברת שם כותב ותוכן לכל פוסט.
- **State:** שימוש ב-`useState` לעדכון התצוגה בזמן ההקלדה.
- **CSS Modules:** עיצוב מבודד ומקצועי לכל רכיב.

## איך מפעילים?
```bash
npm install
npm run dev
